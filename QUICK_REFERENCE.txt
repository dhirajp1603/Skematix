## 8-STEP PIPELINE AT A GLANCE

```
INPUT: 2D Floor-Plan Image
  ↓
1. METRIC NORMALIZATION (1 unit = 1 meter, ~12m width)
  ↓
2. VOLUMETRIC WALLS (0.22m thickness, 1.3m height, open-top)
  ↓
3. FLOOR SLAB (single ground plane at Z=0)
  ↓
4. OPENINGS (doors 0.9×1.1m, windows 0.8×0.5m)
  ↓
5. GEOMETRY CLEANUP (normals, shading, manifold validation)
  ↓
6. MATERIALS (beige walls, gray floor, matte finish)
  ↓
7. CAMERA & LIGHTING (isometric 45°, sun 1.2 energy)
  ↓
8. EXPORT (GLB format, 32 KB, zero-adjustment ready)
  ↓
OUTPUT: Professional 3D Cutaway Model
```

## CONFIGURATION PARAMETERS

Edit `blender/convert_to_cutaway_prod.py`:

```python
# Change these to customize behavior
REFERENCE_BUILDING_WIDTH_M = 12.0      # Target floor width
WALL_THICKNESS_M = 0.22                # Wall thickness
WALL_HEIGHT_CUTAWAY_M = 1.3            # Wall height
FLOOR_THICKNESS_M = 0.12               # Floor thickness
DOOR_WIDTH_M = 0.9                     # Door width
DOOR_HEIGHT_CUTAWAY_M = 1.1            # Door height
WINDOW_SILL_HEIGHT_M = 0.65            # Window sill
WINDOW_WIDTH_M = 0.8                   # Window width
WINDOW_HEIGHT_M = 0.5                  # Window height
COLOR_WALLS_RGB = (0.92, 0.85, 0.74)   # Wall RGB
COLOR_FLOOR_RGB = (0.45, 0.45, 0.48)   # Floor RGB
MATERIAL_ROUGHNESS = 0.75              # Matte (0.0-1.0)
```

## TROUBLESHOOTING QUICK FIXES

| Problem | Quick Fix |
|---------|-----------|
| Flask won't start | `Get-Process python \| Stop-Process -Force` then restart |
| Blender not found | Verify: `blender --version` or set PATH |
| GLB won't open | Check file size > 10 KB, try Blender viewer |
| Wrong scale | Check log for scale factor, verify input image |
| No openings visible | Ensure walls are wide enough (>1.8m) |
| Materials look wrong | Verify COLOR_WALLS_RGB and COLOR_FLOOR_RGB |
| Camera angle off | Adjust camera positioning in Step 7 |
