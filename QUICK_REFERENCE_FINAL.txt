┌─ PIPELINE STAGES ──────────────────────────────────────────────────────────┐
│                                                                             │
│  Stage 1: Semantic Segmentation      → 4-class masks (walls, doors, etc)  │
│  Stage 2: Wall Refinement            → Cleaned wall mask                  │
│  Stage 3: Topology Extraction        → Vertices + edges graph             │
│  Stage 4: Room Detection             → Enclosed rooms (⚠ FAIL FAST #1)    │
│  Stage 5: Metric Normalization       → Pixels to meters (12.0m ref)       │
│  Stage 6: 3D Cutaway Construction    → 3D mesh generation                 │
│  Stage 7: Openings Generation        → Doors + windows cutting            │
│  Stage 8: Comprehensive Validation   → Quality checks (⚠ FAIL FAST #2)    │
│  Stage 9: GLB Export                 → Industry-standard 3D format        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─ SPECIFICATIONS (Hard-coded) ──────────────────────────────────────────────┐
│                                                                             │
│  BUILDING GEOMETRY:                                                        │
│    • Wall thickness:  0.22m (standard brick)                              │
│    • Wall height:     1.3m (open-top cutaway)                             │
│    • Floor thickness: 0.12m (standard slab)                               │
│                                                                             │
│  OPENINGS:                                                                 │
│    • Door size:       0.9m (width) × 1.1m (height)                        │
│    • Window size:     0.8m (width) × 0.5m (height)                        │
│    • Window sill:     0.65-0.80m above floor                              │
│                                                                             │
│  METRICS:                                                                  │
│    • Reference width: 12.0m (±1.5m tolerance)                             │
│    • Scale factor estimated from wall topology                            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
